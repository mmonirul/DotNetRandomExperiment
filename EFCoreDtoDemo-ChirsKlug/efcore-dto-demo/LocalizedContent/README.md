# LocalizedContent Project

## Project Overview
The `LocalizedContent` project is designed to manage and organize localized content efficiently. It provides a structured way to handle various types of content, such as accreditations, descriptions, images, and program fees, through a flexible and extensible model.

## Features
- **CardSet Model**: A central model to group and manage different types of cards.
- **Card Types**: Support for multiple card types, including Accreditation, Description, Image, and Program Fees.
- **Extensibility**: Easily extendable to add new card types or features.
